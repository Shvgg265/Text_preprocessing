# Text_preprocessing
these are some basic preprocessing on text
text is often vectorised to integers.
each word is given a unique index.
e.g. :- I like pizza and I like burger
I - 0
like - 1
pizza - 2
and - 3
burger - 4

so, I like pizza and I like burger : is converted to : 
   [0, 1, 2, 3, 0, 1, 4]
